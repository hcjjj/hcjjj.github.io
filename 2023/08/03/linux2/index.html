网络编程

TCP 服务端

socket() → bind() → listen() → accept() → read()/recv() → write()/send() → close()

TCP 客户端

socket() → connect() → write()/send() → read()/recv() → close()

在使用时，应该检查返回值，以确保操作成功！

UDP 服务器端

socket() → bind() → recvfrom()/sento()

UDP 客户端

socket() → sendto()/recvfrom() → close()

并发服务器模型

  • 方式一：预先创建多个 child 进程等待客户端的连接
  • 方式二：客户端请求到来了再创建并发线程
