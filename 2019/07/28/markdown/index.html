Markdown 常用语法整理

Markdown 是一种可以使用普通文本编辑器编写的标记语言，通过解析器可以使普通文本内容具有一定的格式，更便捷高效地记录思想，整理笔记、知识。

编辑器

各平台都有许多Markdown编辑器客户端，或者使用Web的Markdown编辑器。使用md编辑器可以带来语法高亮、实时预览、快捷补全等功能，比使用普通文本编辑器的体验更好

Typora、Cmd Markdown、Visual Studio Code（安装插件）、各种云笔记软件

        System.out.println("Hello World");
