Visual Studio Code 使用记录

Windows 下配置 C/C++ 环境

  1. 下载并解压 MinGW
  2. Win + R 打开运行，输入 sysdm.cpl，配置 Path 系统变量
  3. 安装 clangd 或 C/C++ Extension Pack 插件
  4. 修改根目录的 .vscode 文件夹中的 tasks.json、launch.json 文件

快捷键修改

VSCode 的 Terminal 中 Ctrl + F 默认为查找，将其修改为 Ctrl + Alt + F，方便在终端中使用 Ctrl + F 进行命令的快速补全

步骤：在 VSCode 中 Ctrl + Shift + P 或者 F1 输入 keyboard 进入 Preferences: Open Keyboard Shortcuts，搜索 Ctrl + F，找到 Terminal: Focus Find 进行修改
