工作区和暂存区

Git 本地的三个工作区域：

  • 工作目录 (Working Directory)

  • 暂存区 (Stage/Index)

  • 资源库 (Repository/Git Directory)

文件的四种状态：

  • Untracked: 未跟踪，不参与版本控制，git add 变为Staged
  • Staged: 暂存状态，git commit 同步到库中
  • Unmodify: 修改后变为Modified，git rm 移出版本库，变为Untracked
  • Modified: 文件已修改，git add 进入staged状态, git checkout 丢弃修改

git push -f 推送到远程仓库并覆盖，谨慎操作
